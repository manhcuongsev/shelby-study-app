# Shelby Study App

A simple application for storing study files on Shelby (Aptos blockchain).

## Setup
1. `pnpm install`
2. `pnpm dev` (run at http://localhost:3000)

## Features
- Upload files to Shelby.

- Receive sharing links (expires if configured).
